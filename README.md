# 해피트리 10STEP 문제은행

영어 시험지 PDF에서 텍스트형 영어 지문을 추출해 STEP 1~10 연습지를 생성하는 정적 웹앱입니다.

## 주요 기능

- PDF.js 기반 브라우저 내 PDF 텍스트 추출
- 추출 지문 복수 선택, 전체 선택 및 전체 해제
- STEP 1~10 선택 생성
- 화면과 PDF의 실제 좌우 2단 시험지 레이아웃
- 선택한 전체 STEP을 하나의 PDF로 저장
- STEP 7 지문별 오류 찾기 문단 및 답안칸 분리

## 사용

`index.html`을 열거나 Render Static Site로 배포합니다.

사진으로 스캔된 PDF는 텍스트 레이어가 없어 별도의 OCR 처리가 필요합니다.

**HappyTree Academy · PDF 업로드 → AI 자동 추출 → 10STEP 연습지 생성**

## 기능
- PDF 시험지 업로드 → Claude AI가 영어 지문 자동 추출
- STEP 1~10 중 원하는 단계만 선택 생성
- A4 인쇄 최적화 출력

## STEP 구성
| STEP | 내용 |
|------|------|
| 1 | 영어 원문 + 해석 |
| 2 | 해석 빈칸 |
| 3 | 영어 빈칸 (명사·형용사) |
| 4 | 해석 쓰기 |
| 5 | 동사 원형 고치기 |
| 6 | 어법·어휘 선택 |
| 7 | 오류 찾기 (3개) |
| 8 | 문장 배열 |
| 9 | 문단 배열 |
| 10 | 영작 연습 |

## 배포
Render Static Site로 배포 — `index.html` 단일 파일
